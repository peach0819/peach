报表逻辑

  1. 默认全局只保留 海拍客 业务. (bu_id = 0)
  2. 默认过滤掉 员工店 (store_type != 9 )
  3. (V2新增)默认过滤掉 美妆门店和伙伴店(store_type!=3 11) 和 SAAS分销订单(sale_dc_id != 0 )
  4. 默认支付时间小于等于方案结束时间
  5. 门店商品的新签时间在方案时间之内
  6. 默认剔除掉卡券票，其他业务域

方案汇总指标逻辑

1、如统计指标为新签门店数： 则不重复计数shop_id,  cout(distinct case when 是否新签成功='是‘ then shop_id else null end )

2、如统计指标为新签商品门店数： 则不重复计数shop_id&item_id  ,  cout(distinct case when是否新签成功='是‘ then shop_id&item_id else null end )

3、如统计指标为新签GMV： 则求和（实货GMV-退款)    sum( case when 是否新签成功='是‘ then 实货GMV-退款  else 0  end )
