



正向：
底层表：自然数据（订单本身的数据，不包含业务逻辑）
数据表：相关业务的订单基础数据，不包含聚合、统计逻辑（订单信息、退款信息都来源底层表）
汇总表：正向方案当月本身的汇总数据（基于人）

dw_order_d -> 1、dw_salary_gmv_rule_public_mid_v2_d（gmv中间表）
           -> 2、dw_salary_gmv_rule_public_d （gmv数据表）
           -> 3、dw_salary_forward_plan_sum_d （正向汇总表）


dw_order_d -> 1、dw_salary_sign_rule_public_mid_v2_d（新签中间表，包含新签品牌、商品数据）
           -> 2、dw_salary_sign_item_rule_public_d （新签商品数据表）
           -> 2、dw_salary_sign_brand_rule_public_d（新签品牌数据表）
           -> 3、dw_salary_forward_plan_sum_d （正向汇总表）


逆向：（除了方案月份当月，别的月份都会跑汇总表）
底层表：自然数据（订单本身的数据，不包含业务逻辑）
数据表：相关业务的订单基础数据，不包含聚合、统计逻辑（历史订单join最新退款）
中间表：逆向方案当月本身的汇总数据（基于人）
汇总表：统计了本月数据和上月数据的 对比信息



dw_order_d -> 1、dw_salary_gmv_rule_public_mid_v2_d（gmv中间表）
           -> 2、dw_salary_gmv_rule_public_d （gmv数据表）
           -> 3、dw_salary_backward_plan_sum_mid_d（逆向中间表）
           -> 4、dw_salary_backward_plan_sum_d（逆向汇总表）


dw_order_d -> 1、dw_salary_sign_rule_public_mid_v2_d（新签中间表，包含新签品牌、商品数据）
           -> 2、dw_salary_sign_item_rule_public_d （新签商品数据表）
           -> 2、dw_salary_sign_brand_rule_public_d（新签品牌数据表）
           -> 3、dw_salary_backward_plan_sum_mid_d（逆向中间表）
           -> 4、dw_salary_backward_plan_sum_d（逆向汇总表）
