电脑本地创建文件夹并共享

  1. 创建共享文件夹

    • 在 Windows 桌面上或你喜欢的任何位置，新建一个文件夹，名字建议用英文，例如 share_ubuntu 。

  2. 启用网络发现和文件共享

    • 打开 控制面板 → 网络和共享中心 → 更改高级共享设置 。

    • 在“专用”网络配置文件下，确保选中以下两项：

      • 启用网络发现

      • 启用文件和打印机共享

    • 点击 保存更改。

  3. 设置文件夹共享

    • 右键点击你刚刚创建的 share_ubuntu 文件夹，选择 “属性” 。

    • 切换到 “共享” 选项卡，然后点击 “高级共享” 按钮 。

    • 在弹出的窗口中，勾选“共享此文件夹” 。

    • （可选）你可以修改“共享名”，保持默认的 share_ubuntu 即可。

    • 点击 “权限” 按钮 。

  4. 设置访问权限

    • 在权限窗口中，默认可能只有 Everyone。为了简单起见，我们可以使用 Everyone 并赋予它读写权限。

    • 选中 Everyone，然后在下方 勾选“完全控制”和“更改”的“允许”复选框 。

    • 点击 “应用” 和 “确定”，一路点击“确定”关闭所有属性窗口。

    小提示： 如果你的公司网络对安全有严格要求，也可以不设置 Everyone，而是点击“添加”，输入你的 Windows 登录用户名，再赋予权限。不过使用 Everyone 在虚拟机测试场景下是最方便的。

  5. 获取 Windows 的 IP 地址

    • 按下 Win + R 键，输入 cmd 并回车，打开命令提示符。

    • 输入 ipconfig 并回车。

    • 在输出信息中，找到你当前使用的网络适配器（例如“以太网”或“WLAN”），记下其中的 IPv4 地址，例如 192.168.x.x。这个地址稍后会在 Ubuntu 上用到。

🐧 第二部分：在 Ubuntu 上访问并复制文件

现在回到你的 Ubuntu 虚拟机，我们将把它连接到 Windows 的共享文件夹。

  1. 安装必要的软件包

    • 打开 Ubuntu 的终端（Terminal）。

    • 执行以下命令，安装 cifs-utils 工具，它能让 Ubuntu 支持访问 Windows 的共享文件夹 ：

      sudo apt update
sudo apt install cifs-utils

  2. 创建挂载点

    • 在 Ubuntu 上，我们需要一个空文件夹作为“挂载点”，把 Windows 的共享文件夹挂载到这里。

    • 执行命令创建一个目录，例如 ~/windows_share：

      mkdir ~/windows_share

  3. 手动挂载共享文件夹

    • 现在，使用 mount.cifs 命令将 Windows 共享文件夹挂载到刚才创建的目录。请将命令中的 //WIN_IP/share_ubuntu 和 username 替换为你自己的信息 。

    • //WIN_IP/share_ubuntu：WIN_IP 替换为你在 Windows 上记下的 IPv4 地址。

    • username：替换为你的 Windows 登录用户名。


    sudo mount -t cifs //10.250.111.240/share_ubuntu ~/windows_share -o username=tao.zheng8833,uid=$(id -u),gid=$(id -g),dir_mode=0755,file_mode=0644

    • 输入命令后回车，终端会提示你输入密码，这里输入你的 Windows 登录密码。

  4. 复制 decoder 文件

    • 现在，你可以像操作本地文件夹一样操作 ~/windows_share 了。

    • 假设你的 decoder 文件在 Ubuntu 的 ~/silk-v3-decoder/silk/ 目录下，执行以下命令复制它：
      bash

      cp ~/silk-v3-decoder/silk/decoder ~/windows_share/

    • 命令执行后，切换回 Windows，打开 share_ubuntu 文件夹，你应该就能看到 decoder 文件了。

  5. 使用完毕后卸载共享（可选）

    • 如果你不再需要连接，可以在 Ubuntu 终端中执行以下命令来断开连接：
      bash

      sudo umount ~/windows_share


然后把生成的decoder文件放到项目下的resource文件夹下即可使用

AmrUtil  展开源码 
package com.yt.touchc.common.util;

import lombok.extern.slf4j.Slf4j;
import ws.schild.jave.process.ffmpeg.DefaultFFMPEGLocator;

import java.io.BufferedReader;
import java.io.FileInputStream;
import java.io.IOException;
import java.io.InputStream;
import java.io.InputStreamReader;
import java.net.URL;
import java.nio.file.Files;
import java.nio.file.Path;
import java.nio.file.StandardCopyOption;
import java.util.Comparator;

/**
 * @author feitao.zt
 * @date 2026/3/18
 */
@Slf4j
public class AmrUtil {

    public static InputStream convertToMp3(String urlString) throws IOException, InterruptedException {
        URL url = new URL(urlString);
        // 1. 创建临时工作目录
        Path workDir = Files.createTempDirectory("wechat_conv_");
        Path inputFile = workDir.resolve("input.amr");
        Path outputFile = workDir.resolve("input.mp3"); // 脚本会生成此文件

        try {
            // 2. 保存微信语音流（跳过头部后的纯数据）到临时文件
            try (InputStream rawInput = url.openStream()) {
                rawInput.skip(1); // 跳过微信私有头
                Files.copy(rawInput, inputFile, StandardCopyOption.REPLACE_EXISTING);
            }

            // 3. 解压 converter.sh 和 silk/decoder
            extractScriptAndDecoder(workDir);

            // 4. 获取 JAVE 内置 FFmpeg 路径
            DefaultFFMPEGLocator locator = new DefaultFFMPEGLocator();
            String ffmpegPath = locator.getExecutablePath();

            // 5. 执行 converter.sh（单个文件模式）
            ProcessBuilder pb = new ProcessBuilder(
                    "bash", workDir.resolve("converter.sh").toString(),
                    inputFile.toString(),  // 输入文件
                    "mp3"                  // 输出格式
            );
            pb.directory(workDir.toFile());
            pb.environment().put("FFMPEG_PATH", ffmpegPath); // 关键：传递自定义 ffmpeg
            pb.redirectErrorStream(true);

            Process process = pb.start();
            // 打印日志（便于调试）
            try (BufferedReader reader = new BufferedReader(new InputStreamReader(process.getInputStream()))) {
                String line;
                while ((line = reader.readLine()) != null) {
                    log.info("[converter] " + line);
                }
            }
            int exitCode = process.waitFor();
            if (exitCode != 0) {
                throw new IOException("转换失败，退出码: " + exitCode);
            }
            return new FileInputStream(outputFile.toFile()) {

                @Override
                public void close() throws IOException {
                    super.close();
                    Files.deleteIfExists(outputFile);
                    Files.walk(workDir)
                            .sorted(Comparator.reverseOrder())
                            .forEach(path -> path.toFile().delete());
                }
            };
        } finally {
            Files.deleteIfExists(inputFile);
        }
    }

    private static void extractScriptAndDecoder(Path workDir) throws IOException {
        // 解压 converter.sh
        Path scriptTarget = workDir.resolve("converter.sh");
        try (InputStream is = AmrUtil.class.getResourceAsStream("/silk-v3-decoder/converter.sh")) {
            if (is == null) {
                throw new IOException("converter.sh not found in resources");
            }
            Files.copy(is, scriptTarget, StandardCopyOption.REPLACE_EXISTING);
        }
        scriptTarget.toFile().setExecutable(true);

        // 创建 silk 子目录并解压 decoder
        Path silkDir = workDir.resolve("silk");
        Files.createDirectories(silkDir);
        Path decoderTarget = silkDir.resolve("decoder");
        try (InputStream is = AmrUtil.class.getResourceAsStream("/silk-v3-decoder/decoder")) {
            if (is == null) {
                throw new IOException("decoder not found in resources");
            }
            Files.copy(is, decoderTarget, StandardCopyOption.REPLACE_EXISTING);
        }
        decoderTarget.toFile().setExecutable(true);
    }
}

#!/bin/bash
# File: converter.sh (modified for custom ffmpeg)
# Date: August 19th, 2016
# Time: 18:56:52 +0800
# Author: kn007 <kn007@126.com>
# Blog: https://kn007.net
# Link: https://github.com/kn007/silk-v3-decoder
# Usage: sh converter.sh silk_v3_file/input_folder output_format/output_folder flag(format)
# Flag: not define   ----  not define, convert a file
#       other value  ----  format, convert a folder, batch conversion support
# Requirement: gcc ffmpeg (but now we use custom ffmpeg via FFMPEG_PATH)

# 支持自定义 FFmpeg 路径（通过环境变量 FFMPEG_PATH 传入）
FFMPEG="${FFMPEG_PATH:-ffmpeg}"

# Colors
RED="$(tput setaf 1 2>/dev/null || echo '\e[0;31m')"
GREEN="$(tput setaf 2 2>/dev/null || echo '\e[0;32m')"
YELLOW="$(tput setaf 3 2>/dev/null || echo '\e[0;33m')"
WHITE="$(tput setaf 7 2>/dev/null || echo '\e[0;37m')"
RESET="$(tput sgr 0 2>/dev/null || echo '\e[0m')"

# Main
cur_dir=$(cd `dirname $0`; pwd)

if [ ! -r "$cur_dir/silk/decoder" ]; then
    echo -e "${WHITE}[Notice]${RESET} Silk v3 Decoder is not found, compile it."
    cd $cur_dir/silk
    make && make decoder
    [ ! -r "$cur_dir/silk/decoder" ] && echo -e "${RED}[Error]${RESET} Silk v3 Decoder Compile False, Please Check Your System For GCC." && exit
    echo -e "${WHITE}========= Silk v3 Decoder Compile Finish =========${RESET}"
fi

cd $cur_dir

while [ $3 ]; do
    # 注释掉进程检查，因为我们的 FFmpeg 是临时路径，不会与其他进程冲突
    # [[ ! -z "$(pidof ffmpeg)" ]] && echo -e "${RED}[Error]${RESET} ffmpeg is occupied by another application, please check it." && exit
    [ ! -d "$1" ] && echo -e "${RED}[Error]${RESET} Input folder not found, please check it." && exit
    TOTAL=$(ls $1|wc -l)
    [ ! -d "$2" ] && mkdir "$2" && echo -e "${WHITE}[Notice]${RESET} Output folder not found, create it."
    [ ! -d "$2" ] && echo -e "${RED}[Error]${RESET} Output folder could not be created, please check it." && exit
    CURRENT=0
    echo -e "${WHITE}========= Batch Conversion Start ==========${RESET}"
    ls $1 | while read line; do
        let CURRENT+=1
        $cur_dir/silk/decoder "$1/$line" "$2/$line.pcm" > /dev/null 2>&1
        if [ ! -f "$2/$line.pcm" ]; then
            $FFMPEG -y -i "$1/$line" "$2/${line%.*}.$3" > /dev/null 2>&1 &
            ffmpeg_pid=$!
            while kill -0 "$ffmpeg_pid"; do sleep 1; done > /dev/null 2>&1
            [ -f "$2/${line%.*}.$3" ] && echo -e "[$CURRENT/$TOTAL]${GREEN}[OK]${RESET} Convert $line to ${line%.*}.$3 success, ${YELLOW}but not a silk v3 encoded file.${RESET}" && continue
            echo -e "[$CURRENT/$TOTAL]${YELLOW}[Warning]${RESET} Convert $line false, maybe not a silk v3 encoded file." && continue
        fi
        $FFMPEG -y -f s16le -ar 24000 -ac 1 -i "$2/$line.pcm" "$2/${line%.*}.$3" > /dev/null 2>&1 &
        ffmpeg_pid=$!
        while kill -0 "$ffmpeg_pid"; do sleep 1; done > /dev/null 2>&1
        rm "$2/$line.pcm"
        [ ! -f "$2/${line%.*}.$3" ] && echo -e "[$CURRENT/$TOTAL]${YELLOW}[Warning]${RESET} Convert $line false, maybe ffmpeg no format handler for $3." && continue
        echo -e "[$CURRENT/$TOTAL]${GREEN}[OK]${RESET} Convert $line To ${line%.*}.$3 Finish."
    done
    echo -e "${WHITE}========= Batch Conversion Finish =========${RESET}"
    exit
done

$cur_dir/silk/decoder "$1" "$1.pcm"
if [ ! -f "$1.pcm" ]; then
    $FFMPEG -y -i "$1" "${1%.*}.$2" > /dev/null 2>&1 &
    ffmpeg_pid=$!
    while kill -0 "$ffmpeg_pid"; do sleep 1; done > /dev/null 2>&1
    [ -f "${1%.*}.$2" ] && echo -e "${GREEN}[OK]${RESET} Convert $1 to ${1%.*}.$2 success, ${YELLOW}but not a silk v3 encoded file.${RESET}" && exit
    echo -e "${YELLOW}[Warning]${RESET} Convert $1 false, maybe not a silk v3 encoded file." && exit
fi
$FFMPEG -y -f s16le -ar 24000 -ac 1 -i "$1.pcm" "${1%.*}.$2" > /dev/null 2>&1
ffmpeg_pid=$!
while kill -0 "$ffmpeg_pid"; do sleep 1; done > /dev/null 2>&1
rm "$1.pcm"
[ ! -f "${1%.*}.$2" ] && echo -e "${YELLOW}[Warning]${RESET} Convert $1 false, maybe ffmpeg no format handler for $2." && exit
echo -e "${GREEN}[OK]${RESET} Convert $1 To ${1%.*}.$2 Finish."
exit
